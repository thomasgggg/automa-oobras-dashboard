# Canteiro — dashboard de controle de obras

## Rodar localmente (opcional, precisa de Node.js instalado)
```
npm install
npm run dev
```

## Publicar de graça
1. Suba esta pasta inteira para um repositório no GitHub.
2. Crie conta em https://vercel.com (pode entrar com o GitHub).
3. Em "Add New Project", selecione este repositório e clique em "Deploy".
4. Abra o link gerado, cole a URL e a chave publishable do seu projeto Supabase quando pedido.
