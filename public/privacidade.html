Política de Privacidade — Viga Automações

Última atualização: julho de 2026.

Quem somos

Viga Automações é um assistente de obras que centraliza fotos, áudios, notas fiscais, documentos e atualizações enviados pelo WhatsApp, organizando-os por obra em um painel de consulta.

Quais dados coletamos

Coletamos as mensagens, fotos, áudios e documentos que você ou sua equipe enviam voluntariamente pelo WhatsApp para o número conectado a este serviço, além do número de telefone do remetente, para organizar os registros por obra.

Como usamos os dados

Os dados são usados exclusivamente para registrar, organizar e exibir as informações da obra no painel de consulta, e para enviar alertas automáticos sobre orçamento e prazo pelo WhatsApp.

Compartilhamento

Não vendemos nem compartilhamos seus dados com terceiros, exceto os provedores de infraestrutura necessários para operar o serviço (armazenamento de banco de dados e hospedagem).

Exclusão de dados

Para solicitar a exclusão dos seus dados, entre em contato pelo mesmo número de WhatsApp usado para os registros.

Contato

Dúvidas sobre esta política podem ser enviadas para o e-mail informado no aplicativo Meta for Developers associado a este serviço.
